Intoduction

This API provides users with information about different movies, director, and genres.Users will be able to sign up, update their personal information, and create a list of their favorite movies.

myFlix API References







REQUEST URL HTTP Method Request body data format Response body data format
Get a list of available movies /movies GET None A JSON object holding data about all the movies
Example: [{ Title: "", Director: {Name: "",Bio: "", Birthdate: "", } "", Genre: { Name: "", Description: "..",}, ]
Get data about a movie by title /movies/:title GET None A JSON object holding data about a single movie by title. Description, genre, director image URL, whether its featured or not,
Example:
[{Title: "Don't Look up", Director: {Name: "Adam McKay",Bio: "Adam McKay is an American...", Birthdate: "1967", Genre: { Name: "Science-Fiction", Description: "blablalbalbal...", }]
Get data about genre /movies/genre/:genreName GET None A JSON object holding data about genre (name, description)
Example:
{ Name: "Drama", Description: " blablabal"}
Get data about director by name /movies/director/:directorName GET None A JSON object holding data about director,
Example:
{ Name: "John Doe", bio: "blablabal", birth: "1900", death: "2021"}
Allow new user to register /users POST A JSON object holding data about the user to register,
structured like:
{ name: "Kim", favoriteMovies: [] } 		A JSON object holding data about registered user with ID,
Example:
{ name: "Mandela", favoriteMovies: [], id: "a12dhgd" }
Allow new user to login /users/login POST A JSON object holding data about the user to login,
structured like:
{ Username: "user2", Password: "452523455", Email: "ghghgh@whatever.com" } 		A JSON object holding data about the user with ID and token
Example:
{ "user": { "_id": "61f3c5efe1aa41dc1f9bc9ef", "Username": "user2", "Password": "$2b$10$LzD12sYfuJ5vp04j9uIWh.SzjbHarCTyJlXSw8cwXql725m2gR1Jm", "Email": "xyzzzz@gmail.com", "Birthday": "2000-01-01T00:00:00.000Z", "FavoriteMovies": [], "__v": 0 }, "token": "...JhbGciOi1NiIsInRmFtZS.." }
Allow registered users to update their info /users/:Username PUT A JSON object holding data about the user to be updated .
Example: { Username: String, (required) Password: String, (required) Email: String, (required) Birthday: Date } 		A JSON object holding updated data about the user,
structured like:
{ _id: "String", Username: "Mandela", Password:"23323" favoriteMovies:[],},
Get data about a registered user by userName /users/:userName GET Example: users/John%20Doe A JSON object holding data about registered user
Allow registered users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST Example: /users/John%20Doe/movies/12323232 A JSON object holding data about the updated user's favorite movie list
Allow registered users to delete a movie from their list of favorites /users/:Username/FavoriteMovies/:MovieID DELETE None A JSON object holding data about the updated user's favorite movie
Allow existing users to deregister /users/:userName DELETE None Null
