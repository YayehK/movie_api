Intoduction

This API provides users with information about different movies, director, and genres.Users will be able to sign up, update their personal information, and create a list of their favorite movies.

myFlix API References






REQUEST URL HTTP Method Request body data format Response body data format
Get a list of available movies /movies GET None A JSON object holding data about all the movies
Example: [{ Title: "Don't Look up", Director: "Adam McKay", Genre: { Name: "Science", Description: "blablalbalbal...",}, ]
Get data about a movie by title /movies/:movieTitle GET None A JSON object holding data about a single movie by title. Description, genre, director image URL, whether its featured or not,
Example:
[{Title: "Don't Look up", Director: "Adam McKay", Genre: { Name: "Science", Description: "blablalbalbal...", }]
Get data about genre /movies/genre/:genreName GET None A JSON object holding data about genre (name, description)
Example:
{ Name: "Drama", Description: " blablabal"}
Get data about director by name /movies/director/:directorName GET None A JSON object holding data about director,
Example:
{ Name: "John Doe", bio: "blablabal", birth: "1900", death: "2021"}
Allow new user to register /users POST A JSON object holding data about the user to register,
structured like:
{ name: "Kim", favoriteMovies: [] } 		A JSON object holding data about registered user with ID,
Example:
{ name: "Mandela", favoriteMovies: [], id: "a12dhgd" }
Allow registered users to update their info /users/:userName PUT A JSON object holding data about the changed userName.
Example: { name: "Mandela", } 		A JSON object holding updated data about the user,
structured like:
{ name: "Mandela", favoriteMovies: [], id: "a12dhgd" },
Get data about a registered user by userName /users/:userName GET Example: users/John%20Doe A JSON object holding data about registered user
Allow registered users to add a movie to their list of favorites /users/userName/:movieTitle POST Example: /users/Joe/Encanto A text message: "movieTitle" has been added to favorites
Allow registered users to delete a movie from their list of favorites /users/:userName/:movieTitle DELETE None A text message: "movieTitle" has been removed from [username]'s favorite
Allow existing users to deregister /users/:userName DELETE None A text message: [userName]'s account was succesfully deleted
